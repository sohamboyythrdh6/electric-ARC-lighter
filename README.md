# electric-ARC-lighter
an electric lighter is a very useful project. which can be use for start a fire or ignite our gas stove .it's not like other lighters because it is charge by a type c charger and it dosen't need to refill gas .just charge and whenever need just press the switch and use.this project will teach us how crazy and dangerous can a hihg voltage transformer is which can turn 5 volt into 1000 of volts.

to build this lighter you jest need to collect the components pcb and those 3d models then connect all the parts to the pcb using solder later confirming everypart is in correct place put everything in the case and glued it and then it will be ready to use .But before using full charge this.
